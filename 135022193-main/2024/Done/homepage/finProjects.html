Blink Detection

My first real project that I finished was a blink detection program.
This program was the first step to my Elden Ring Project.

This Project used the OpenCV library from Intel and Python.
It's surprisingly simple, all it does is first of all detect a face, if it has found a face
it will start trying to detect eyes, if it detects 2 eyes it will wait for both eyes to be closed,
once it detects that both eyes are closed it will add 1 count to the blink counter
and it will wait for both eyes to be open again before it starts to look for a blink again.
