Discount Checker

Discount Checker is a website ive wanted to make for quite a while now (and it will be my final project for CS50).
It is a website where you can put in the link of a product from a popular webshop.
The website will then show the original price of the product and (if on discount) the discounted price of the product.
My plan is to make it give you notifications when something is on discount and keep track of prices to check for an all time low.

The reason i want to make this website is ofcourse to learn more about web development
but also because i have wanted a website or app like this for a while, so i'll make it myself.

The Elden Ring Project

Now for a more hobby oriented and fun focused one. The Elden Ring Project I have worked on for a bit.
This project has no real use outside of it being fun.
For a while now I have seen peopple complete the videogame Elden Ring in bizarre ways with weird controllers
and I have always wanted to make one of those myself, so I came up with the blink to roll concept.
The basic concept is a blink detector that will send a signal to roll to the game when you blink,
this way the game will have be played by blinking instead of just pressing the button.

I have gotten pretty far with the project so far, I got the blink detection and managed to make it roll when I blink.
But its not optimised and if i try to connect it to a virtual controller it crashes. So my plan is to put the project on pause for now
and learn more about Arduino's and Raspberry Pi's. The reason for this is that I believe i can run the blink detection through a Raspberry Pi
which will take the load of that off of my computer, which will then be able to smoothly run everything.
